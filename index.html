Your performance just here...

A CPU (Central Processing Unit) is the "brain" of a computer, a crucial piece of hardware that executes instructions from software, manages data, and performs calculations, making it responsible for running the operating system and all applications. It acts as the central control center, directing other components and processing user commands to turn input into output, found in everything from PCs and laptops to smartphones.

A GPU (Graphics Processing Unit) is a specialized electronic circuit that rapidly manipulates memory to create images for display, excelling at parallel processing tasks like gaming, video editing, AI, and scientific computing by handling many calculations at once, unlike a CPU which handles fewer, more complex tasks. GPUs can be integrated into a system's motherboard or exist as powerful, separate "discrete" cards, accelerating visual and data-intensive workloads far beyond basic graphics.

HDD stands for Hard Disk Drive, a traditional data storage device that uses spinning magnetic platters and read/write heads to store and access digital information (like your operating system, apps, and files) non-volatilely, meaning data remains even when the power is off, making it a reliable choice for long-term storage.

RAM (Random Access Memory) is your computer's short-term, high-speed, temporary memory, holding data for currently running applications and files so the CPU can access them instantly, allowing for smooth multitasking and fast performance; it's volatile, meaning data disappears when the power is off. Think of it as your desk space where you keep papers you're actively working on, unlike long-term storage (like a hard drive or SSD) where things are filed away.

A motherboard is the central printed circuit board (PCB) in a computer that acts as the main communication hub, connecting and allowing vital components like the CPU (processor), RAM (memory), graphics card, and storage drives to work together and communicate with each other and external peripherals (like keyboards, monitors, and speakers). It serves as the foundation, providing power and data pathways, essentially the "backbone" or "skeleton" of the entire system.

Power Control Unit, a dedicated component managing power and thermal aspects of a processor or system for efficiency, or Packet Control Unit, crucial in older mobile networks (GPRS) for managing data flow between radio and core networks. It can also refer to Personal Computing Unit in specialized industrial contexts or even Performance Cost Unit in video games to limit performance impact.

A monitor is an essential computer output device that displays visual information like text, images, and videos from the computer's video card onto a screen, allowing users to see and interact with the system's processes, much like a TV but usually with higher resolution for sharper detail. Modern monitors use flat-panel technology (like LCD with LED backlights) instead of older Cathode Ray Tubes (CRTs) and connect via cables such as HDMI or DisplayPort
